# Warrior attack simulator
## Description 
A python grid-based simulation that checks whether warrior pieces on an nxn board can attack each other based on a movement pattern of two steps horizontally or vertically followed by one diagonal step
## How it works
- The board is nxn in size
- 'W' represents a warrior
- '.' represents an empty space
- The program checks if any warrior can attack another warrior using the movements rules 
## Movement rule
A warrior moves 
- 2 steps vertically or horizontally
- then 1 step diagonally
## How to run
1. run the program using python
2. Enter a board size(n)
3. Enter the board as input
## Output 
The program prints whether warriors can attack wach other or not
